Introdução ao CSS

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é uma linguagem de estilo utilizada para definir a apresentação de documentos HTML. Ele permite separar a estrutura de um site da sua aparência, facilitando a manutenção e tornando o processo de desenvolvimento mais organizado.

O que o CSS faz?

CSS é responsável pela parte visual dos sites e páginas da web. Com ele, podemos modificar:

  • Fontes: Definir tipos de letra, tamanhos, cores, estilos, etc.
  • Layout: Organizar os elementos na página, como posicionamento, margens e espaçamentos.
  • Cor: Alterar as cores de texto, fundo, bordas e outros elementos.
  • Animações e transições: Criar efeitos visuais, como transições suaves ou animações de elementos.

Como o CSS funciona?

O CSS aplica-se aos elementos HTML de uma página através de seletores. Um seletor pode ser uma tag HTML, uma classe ou um ID. Através desses seletores, podemos alterar as propriedades de estilo dos elementos selecionados.

Por exemplo, para mudar a cor de um título (h1) em uma página, podemos usar o seguinte código CSS:

h1 { 
        color: blue;
    }

Conclusão

O CSS é uma ferramenta essencial no design de páginas web modernas. Ele oferece uma infinidade de opções para personalizar o estilo e a aparência de um site, permitindo que ele seja visualmente atraente e funcional.
